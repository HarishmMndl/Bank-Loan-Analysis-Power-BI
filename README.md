# Bank-Loan-Analysis-Power-BI

This project focuses on analyzing loan data to identify patterns behind loan defaults and understand how different factors like income, employment type, and age group impact repayment behavior. The dashboard provides clear, interactive insights to support data-driven decision-making in financial risk analysis.

 ##Key Objectives
* Analyze default rate trends over time (Year-on-Year)
* Identify high-risk segments based on:
* Income brackets
* Employment type
* Age groups
* Understand distribution of loan amounts and borrower profiles
* Build an interactive dashboard for quick insights
## Key Insights
* Higher default rates observed in specific income segments
* Employment type plays a significant role in repayment behavior
* Certain age groups show higher financial risk
* Loan amount distribution varies significantly across categories
  ## DAX Functions Used

This project heavily utilizes DAX for dynamic calculations and insights:

1. CALCULATE() – Context transformation for accurate metrics
2. FILTER() – Conditional data filtering
3. DIVIDE() – Safe division handling (avoiding errors)
4. COUNTROWS() – Counting total and filtered records
5. ALLEXCEPT() – Maintaining grouping context
6. SWITCH() – Creating income brackets dynamically
7. AVERAGEX() – Calculating average loan amounts across groups
8 VAR – Improving performance and readability of measures
## Key Measures Created
1. Default Rate by Year (YoY Analysis)
2. Default Rate by Employment Type
3. Income Bracket Classification
4. Average Loan Amount by Age Group
## Tools & Technologies
* Power BI (Data Visualization)
* DAX (Data Analysis Expressions)
* Excel / CSV (Data Source)
